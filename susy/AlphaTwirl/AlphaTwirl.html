Instructions

On lxplus

Requires CMSSW to retrieve latest version of python and installation of several R packages

  • cmsrel CMSSW_7_5_0

    cd CMSSW_7_5_0/src

    cmsenv

    git clone -b v0.6.0 git@github.com:CMSRA1/AlphaTwirl.git

    git clone -b dev-ops git@github.com:CMSRA1/alphat-scripts.git scripts

    source scripts/setup.sh

    git clone -b v0.1.1 git@github.com:TaiSakuma/mianRs.git

    • Edit ./scripts/print_commands.sh

    If you used AlphaProcess to transfer the trees, then please use the destination directory of the copy here
