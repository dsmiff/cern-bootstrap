Guide

HCal has two L1 Keys, 1 for the magnet at 3.8T and the other for 0T. If the magnet changes state, need to confirm with the HCal DOC that this change of key is okay and change the TSC keys and l1-hlt respectively.
The L1_HLT key contains 3 subkeys:

  • TSC Key : Steers most static configuration parameters of the trigger system hardware and software
  • GT Run Settings Key : Enables/ disables trigger bits and sets prescale factors
  • HLT Key : Sets the HLT parameters, paths and prescale factors

    • To update the L1 and HLT keys together follow this link.
    The list on Trigger Modes can be found here.

    The HLT key is usually provided by the HLT DOC. The others, TSC_KEY and GT_RS_KEY can be modified here with instructions. and here.
